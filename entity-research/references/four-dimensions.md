# 四维分析框架详解

## 维度一：本体 — 它是什么？

**核心问题**：静态结构分析，是一切判断的基石。

**最大风险：粉饰**。要看有什么，更要看缺什么。官网半年没更新、LinkedIn履历避重就轻、投资机构没有清晰LP背景——缺失本身就是信号。

### 个人本体分析框架

| 子维度 | 要看什么 | 信号解读 |
|--------|----------|----------|
| 基础身份 | 当前职位、过往所有经历、教育背景 | 跳槽逻辑是否清晰？有无断层？ |
| 能力成就 | 硬技能（专利/资质）、可验证成就（融资/上市/退出） | 自我宣称与实际履历是否一致？ |
| 社会身份 | 政协/行业协会/大赛评委等背书 | 第三方背书的可信度？ |
| 内容输出 | 在哪里输出什么内容，频率和质量 | 原创还是搬运？是否有深度？ |

**不要只看正面**：PR能力强的人，"希望被你看到的样子"会很完美。分析时主动寻找矛盾点。

### 机构本体分析框架

| 子维度 | 要看什么 | 信号解读 |
|--------|----------|----------|
| 法律身份 | 注册名称、股权结构、实际控制人 | 谁真正控制这家机构？ |
| 核心产品 | 价值主张、主要业务、收入来源 | 商业模式是否可持续？ |
| 团队构成 | 规模、背景、关键人物 | 机构能力是否依赖某个人？ |
| 运营数据 | 基金规模/投资数量/上市记录等可量化成果 | 宣称与实际是否匹配？ |

## 维度二：动力 — 它为何而动？

**核心问题**：驱动系统分析。理解动机 → 预测行为 → 找到价值共赢点。

**这是最核心的维度**：动机决定了能否共渡难关，决定了合作是否值得长期投入。

### 动机层次模型

**表层**（对方说的）→ **中层**（可推断的）→ **深层**（真实驱动力）

| 表层宣称 | 可能的深层动机 | 验证方法 |
|----------|----------------|----------|
| "推动产业创新" | 成为生态规则制定者 | 看其历史决策是否倾向掌控节点 |
| "支持创业者" | 快速积累项目量完成KPI | 看投资条款是否苛刻 |
| "行业认可" | 上榜单、媒体曝光、地位焦虑 | 看其在公开场合的发言频率和主题 |
| "求知探索" | 纯粹好奇心 | 看其在无利益场景下的参与度 |

### 资源链路分析

三个关键问题：
- **资源从哪来**？→ 对谁负责，受谁约束
- **资源怎么转化**？→ 真正在意什么，花钱重心在哪
- **资源去哪里**？→ 最终产出是什么，怎么衡量成功

### 激励机制分析

**关键判断**：个人成功与组织成功是否对齐？

典型不对齐案例：
- 投资经理KPI是投资数量 → 倾向多投，不在乎后续
- 孵化器KPI是政府指标 → 倾向覆盖数量，不在乎PMF
- 基金合伙人Carry来自退出 → 倾向short-term可见价值

对这些不对齐的理解，能找到"个体利益切入点"——有时通过解决个体问题比谈机构合作更有效率。

### 压力与约束

| 类型 | 内容 | 对合作的影响 |
|------|------|-------------|
| 时间压力 | deadline、募资周期、考核节点 | 此时合作意愿最高 |
| 资源瓶颈 | 缺项目流、缺人才、缺特定资源 | 你能补什么最重要 |
| 外部压力 | LP要求、竞争对手崛起、政策变化 | 影响其风险偏好 |
| 能力刚性 | 被过去成功经验束缚 | 新方向的接受度会低 |

## 维度三：连接 — 它处于怎样的生态位？

**核心问题**：关系网络分析。揭示力量来源与脆弱性。

### 网络拓扑三类型

| 类型 | 特征 | 战略价值 |
|------|------|----------|
| **强连接** | 深度依赖，高频互动 | 稳定性来源；可作为资源跳转节点 |
| **弱连接** | 松散广泛，偶尔接触 | 新信息、新机会的主要来源 |
| **结构洞** | 桥接两个不相连的群体 | 最高价值位置，信息优势+控制权 |

### 关键分析

**它连接了谁** → 潜在的资源价值；BD时可以"经由它跳转"到你需要的资源

**它没有连接谁** → 可能是系统性风险（脆弱点），也可能是你的结构性机会（你能成为它需要的连接）

**它是否占据结构洞** → 评估其信息优势和谈判筹码

### 个人网络分析

- **核心圈**：Mentor、联合创始人、"私人董事会"3-5人
- **专业网络**：强连接同行、弱连接熟人
- **影响力层**：影响他的人（他转发/崇拜谁）、被他影响的人

### 机构网络分析

- **资本联盟**：LP背景、合投伙伴派系
- **竞争格局**：直接竞争者、替代者、互补者
- **生态依赖**：项目源、人才源、退出路径

## 维度四：演化 — 它将走向何方？

**核心问题**：动态发展预测。支持合作时机判断。

**关键洞察**：在关键转型期和低谷期的合作往往更容易达成，且更具战略价值。

### 历史路径依赖

- **创始神话**：创立故事中的核心挑战与解决方式 → 固化为"肌肉记忆"
- **关键转折**：重大成功、失败、危机应对 → 塑造今天的文化
- **沉没成本**：是否被过去的巨大投入绑架，无法转向

### 当前适应性

- **学习速度**：从市场反馈和失败中调整有多快？
- **实验能力**：有没有健康的内部试错？
- **趋势响应**：对AI等大趋势是引领者/适应者/抗拒者？

### 未来轨迹预测

| 路径 | 描述 | 触发条件 |
|------|------|----------|
| 基线 | 按当前模式自然发展 | 无重大变量 |
| 最优 | 所有积极因素叠加 | 抓住某个关键机会 |
| 最差 | 关键风险爆发 | 核心人才离职/政策变化/市场下行 |
| 引爆点 | 进入新阶段的标志性事件 | 值得持续追踪的信号 |

### 生命周期判断

对人：快速上升期 / 平台巩固期 / 功成身退期
对机构：野蛮生长期 / 精细运营期 / 创新者窘境

**不同阶段的合作策略完全不同**：上升期的人想要资源和验证，巩固期的人想要稳定，身退期的人想要legacy。

## 决策公式（投资/孵化/深度合作场景）

一个值得长期合作的对象，必须同时满足：
1. 四个维度都过及格线
2. 至少一个维度卓越
3. 任何维度都没有致命伤

| 组合 | 判断 |
|------|------|
| 卓越本体 + 健康动力 | 极品，优先 |
| 平庸本体 + 强演化能力 | 潜力股，长期持有 |
| 强大本体 + 扭曲动力 | 定时炸弹，回避深度合作 |
| 任何维度有致命伤 | 无论其他多好，暂停 |
