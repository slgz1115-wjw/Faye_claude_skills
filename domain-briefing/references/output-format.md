# 文档输出格式规范

## 基本原则

最终文档的标准是：**一个完全不了解这个领域的聪明人，读完之后能建立结构化认知，并看清这个机构/人的运作逻辑。**

不是百科全书，是「聪明人的快速入门 + 第一手洞察」。

---

## docx生成调用流程

使用前必须先读取 `/mnt/skills/public/docx/SKILL.md`，严格按照其中的规范生成。

关键参数：
- 页面尺寸：A4（11906 × 16838 DXA）
- 页边距：四边各1440 DXA（1英寸）
- 字体：Arial
- 颜色主题：深蓝（#1F4E79）主标题，中蓝（#2E75B6）二级标题

---

## 文档视觉结构规范

### 标题层级使用

```
H1（大标题）：报告的四个主要部分
H2（中标题）：每个部分下的主要模块
H3（小标题）：模块内的具体分析点
```

H1使用深蓝色 + 底部边框线，视觉上形成清晰分隔。
H2使用中蓝色，无边框。
H3使用深灰色粗体。

### 必用的三种视觉元素

**1. callout box（信息框）**
用于：来自录音的重要原话引用、关键数据汇总、核心结论
格式：有颜色填充的表格，标题行深色，内容行浅色

**2. 引言块（quote box）**
用于：录音原话引用（左侧蓝色竖线 + 浅灰背景 + 斜体）
一定要和引用后的分析成对出现

**3. 双色对照表**
用于：概念对比、双方价值互补矩阵、客户分层
表头行深蓝色白字，内容行交替浅蓝/白色

### 段落规范

- 正文段落：不超过5行，超过就拆分或用列表
- 列表项：每项至少1句完整的话，不要只写关键词
- **不使用**Unicode bullet（•）→ 使用docx numbering config
- **不使用** `\n` 换行 → 使用独立Paragraph元素

## 各章节内容规范

### 封面页

```
文档大标题（36-40pt，深蓝色）
副标题：聚焦机构/人名（24pt，中蓝色）
---
文件说明（12pt，灰色正文）：
  - 数据来源：[extended search调研] + [XXXX年X月X日会谈录音转写]
  - 调研对象：[人名/机构名]
  - 内容范围：[领域名]核心概念体系 + 机构运作逻辑 + 合作分析
```

### 第一部分：概念体系

每个概念的标准结构：
```
H3：概念名
正文：用一句话说清楚它是什么（不是教科书定义，是投资人需要理解的角度）
callout box：关键数据或来自录音的一手解释
正文：这个概念对孵化/投资判断的实际意义
```

**重要**：概念的顺序要从最基础的开始，有逻辑依赖关系（后面的概念依赖前面的）。

### 第二部分：行业现状

必须包含的数据维度：
- 规模/增速（要有具体数字，有来源）
- 中国vs全球对比（如果有差距，解释原因）
- 近期政策变化（特别是2024-2025的新动向）
- 核心痛点（2-3个，要有深度，不是列清单）

### 第三部分：机构/人物背调

**机构部分**必须回答：
- 它靠什么挣钱？（核心收入来源）
- 它的真实竞争优势是什么？（不是PR说的那种）
- 它在行业生态里处于什么位置？
- 它的局限性是什么？

**人物部分**必须回答：
- 他的职业路径说明了什么？（跳槽/晋升/创业的逻辑）
- 他真正擅长什么？
- 他的价值观/判断框架是什么？（从录音原话里提取）
- 他对什么事情/什么类型的人感兴趣？

**案例部分**：
- 选录音中提到的最具体的案例（有数字、有结果的）
- 用callout box单独呈现
- 分析：这个案例揭示了什么运作逻辑

### 第四部分：合作分析

**价值互补矩阵**（双列表格）：
```
左列：对方/该机构能提供什么
右列：潜在合作方能提供什么
```
每行对应一个具体的价值维度，不要空洞。

**可落地的合作形式**（用callout box，每种模式一个box）：
- 模式名称
- 具体内容（做什么，谁做，怎么触发）
- 预期价值（对双方各什么好处）

**合作信号**（来自录音的直接证据）：
- 引用原话
- 分析信号强度（明确表达 vs 隐含意向）

### 附录：概念速查表

双列表格：
- 左列：概念名（粗体，浅蓝色底）
- 右列：一句话定义（不超过30字）

所有在正文中出现过的专业术语都要收录。

## 文档长度参考

| 部分 | 目标长度 |
|------|---------|
| 封面+说明 | 半页 |
| 第一部分（概念体系） | 3-5页 |
| 第二部分（行业现状） | 2-3页 |
| 第三部分（机构/人物） | 4-6页 |
| 第四部分（合作分析） | 2-3页 |
| 附录 | 1-2页 |
| **总计** | **12-20页** |
