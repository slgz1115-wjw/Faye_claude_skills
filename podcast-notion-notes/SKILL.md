# Podcast → Notion 结构化笔记 Skill

## 适用场景

用户上传播客逐字稿（.docx），提供 shownotes 时间轴，要求提炼某位嘉宾观点写入 Notion 指定文件夹/页面。

---

## 核心流程（严格按顺序执行）

### Step 1：读取文件

```bash
pandoc /mnt/user-data/uploads/<filename>.docx -o /home/claude/transcript.md
wc -l /home/claude/transcript.md
```

然后分段读取全文（每次 view 600 行左右），**必须读完所有内容**，不得跳过中间段落。

---

### Step 2：识别说话人

- 文档中通常标注为「发言人1」（主播）和「发言人2」（嘉宾）
- 默认提炼**说话人2（嘉宾）**的观点
- 如用户指定其他说话人，以用户指定为准

---

### Step 3：内容提炼（三个必出模块）

#### 模块A：反共识金句（页面开头，作为索引）

- 从全文中提炼 10-20 条反共识、高密度金句
- 格式：引号包裹，每条独立一行
- 标准：不是「说得好听」，而是「与常识相悖但有完整论证支撑」
- 放在正文最前面，作为整篇笔记的快速入口

#### 模块B：分章节核心观点（按 shownotes 时间轴组织）

每个 shownotes 条目对应一个二级标题章节，内容包含：

1. **核心命题**（1-2句话，该章节的中心论点）
2. **展开论述**（包含前因后果、推导链、举例、与其他章节的内部关联）
3. **话语体系**（该嘉宾特有的概念/框架/比喻，直接引用原文用词）

**要求**：
- 不得简化为要点列表了事，必须还原论述的完整结构
- 如果嘉宾在该话题上的论述有多个层次（比喻→推理→实践），都要体现
- 保留嘉宾的原话表达风格（不要翻译成四平八稳的商业语言）

#### 模块C：专业名词解释词典（页面末尾）

从全文中识别所有有理解门槛的概念，包括：

- 经济学/社会学效应（马太效应、长尾效应等）
- 数学/物理概念（贝叶斯公式、奥卡姆剃刀等）
- 哲学概念（存在主义、荒诞主义等）
- 行业术语（prompt engineering、向量数据库等）
- 嘉宾自创概念（需标注「嘉宾定义」）

每个词条格式：
```
**[名词]**（[英文/原文]）
[出处/学科来源]。[通俗解释，2-3句话，说清楚它是什么、为什么重要、在本集播客中如何使用]
```

示例：
> **马太效应**（Matthew Effect）
> 出自《圣经·马太福音》，现代社会学概念。指强者愈强、弱者愈弱的正反馈结构——资源、机会、声誉往往向已占优势的一方进一步集中，形成「赢家通吃」局面。本集中用于描述互联网平台竞争的底层规律：谁先过临界节点数，谁的网络价值就呈指数领先，后来者几乎无法追赶。

---

### Step 4：写入 Notion

**必须写入指定父页面**，不得创建孤立页面。

#### 确认父页面 ID

如果用户已提供父页面名称或链接：

```
使用 notion-search 工具搜索该页面名称，获取 page_id
```

如果用户未提供，创建前**必须询问**：
> 请告诉我你想放到 Notion 哪个文件夹/页面下？（提供页面名称或链接均可）

#### 页面创建参数

```python
notion-create-pages(
  parent={"page_id": "<目标父页面ID>"},
  pages=[{
    "properties": {"title": "<播客名称> | <嘉宾名> 观点提炼"},
    "icon": "🎙️",
    "content": "<完整内容，包含三个模块>"
  }]
)
```

#### 页面标题规范

格式：`{播客节目名} {期号}｜{嘉宾名}观点提炼：{主题}`

示例：`无人知晓 E45｜李继刚观点提炼：人何以自处`

---

## 质量标准

| 检查项 | 标准 |
|--------|------|
| 覆盖率 | shownotes 每个条目都有对应章节，无遗漏 |
| 深度 | 每章节不少于 200 字，有论述结构而非列表 |
| 反共识金句 | 不少于 10 条，有真正的反常识张力 |
| 名词词典 | 覆盖全文所有有理解门槛的概念，不遗漏 |
| Notion 归档 | 写入正确父页面，不是孤立页面 |
| 忠实度 | 不混入提炼者自己的观点，不翻译原话风格 |

---

## 常见错误，避免

- ❌ 把章节写成要点 bullet list 了事（要有完整论述）
- ❌ 把嘉宾的话翻译成「官方语言」（保留原有风格和自创词）
- ❌ 没读完全文就开始提炼（必须 view 完所有行）
- ❌ 创建孤立 Notion 页面（必须指定 parent page_id）
- ❌ 名词词典漏掉嘉宾自创概念（自创概念最有价值）
- ❌ 反共识金句选了「听起来好听」的（要选「反常识但有论证」的）
